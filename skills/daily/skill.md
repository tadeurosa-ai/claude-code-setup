# /daily — Resumo do Dia

Gera um resumo estruturado do dia de trabalho.

## O que faz

1. Lista projetos com atividade recente (últimas 24h)
2. Mostra tarefas abertas por projeto
3. Sugere próximos passos baseado no contexto

## Uso

```
/daily
```

## Saída esperada

```
## Daily — [data]

### Projetos ativos
- [projeto]: [status atual]

### Tarefas abertas
- [ ] item 1
- [ ] item 2

### Próximos passos
1. ...
```
